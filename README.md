# Restaurant Ordering System

A Computer Networks mini project implementing a secure client-server based restaurant ordering system using Python sockets and SSL/TLS.

## Features
- Client-server communication using sockets
- Secure communication using SSL/TLS
- Display restaurant menu
- Place food orders
- Calculate total bill with 5% GST
- Server-side order summary display

## Project Structure
- `client.py` - Client-side application
- `server.py` - Server-side application
- `server.crt` - SSL certificate
- `server.csr` - Certificate signing request
- `docs/Project_Report.pdf` - Project report
